修改博客配置：
  编辑 `_config.yml` 可修改博客标题、描述、作者等信息：
  ```yaml
  title: 你的博客标题
  subtitle: 副标题
  description: 博客描述
  author: 你的名字
  language: zh-CN  # 设置为中文
  timezone: Asia/Shanghai  # 设置时区
  ```

 2. 优化博客主题

如果你使用的是 NexT 主题（日志中显示 `NexT version 8.23.1`），可以：
- 编辑主题配置文件 `themes/next/_config.yml` 调整样式
- 开启评论功能（如 Gitalk、Utterances）
- 配置社交链接、阅读统计等功能

 3. 绑定自定义域名（可选）

如果需要使用自己的域名（如 `sutang.com`）：
1. 在域名服务商处添加 DNS 解析（CNAME 指向 `sutang-vain.github.io`）
2. 在 Hexo 项目 `source` 目录创建 `CNAME` 文件，内容为你的域名
3. 重新部署：`hexo clean && hexo g && hexo d`
4. 在 GitHub 仓库 Settings → Pages 中启用 HTTPS
